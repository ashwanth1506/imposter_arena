# Imposter Arena

A fast-paced social-deduction arena where players compete, cooperate, and try to outwit each other. "Imposter Arena" provides a flexible foundation for building multiplayer rounds with hidden roles, objectives, and customizable arenas.

> NOTE: This README is a general template. If you want commands and examples tailored to the repository's actual tech stack (Unity, Node.js, Python, Rust, etc.), tell me the main language/framework and I will update the file.

## Features

- Round-based multiplayer matches
- Hidden roles (imposters and crewmates) and role-based objectives
- Lobby and matchmaking support (local or online)
- Custom maps / arenas and game modes
- AI bot support for solo testing
- Extensible codebase for adding new mechanics
